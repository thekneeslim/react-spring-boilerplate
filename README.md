
## Boiler plate React with SpringBoot Backend

Frontend:   React created using `create-react-app`.
Backend:    SpringBoot 2.1.2.RELEASE with Maven

## Dev Environment
```
mvn spring-boot:run
cd frontend
npm start                   
```

## Running Tests

#### React
```
cd frontend
npm test
```

#### Java
```
mvn test
```


#### Project Build
```
mvn clean install
```
